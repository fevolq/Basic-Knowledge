1.插入数据
  1.1 全部插入
        insert into 表名 values(值1，值2，值3，・・・); ――对应字段进行插入
  1.2 指定插入
        insert into 表名(字段1，字段2，・・・) values(值1，值2，・・・); ――指定对哪些字段进行插入，插入的值与前面的字段相对应
  1.3 插入多条记录
        insert into 表名 values(值1，值2，・・・),(值1,值2,・・・),・・・; ――一个括号内是一条记录
  1.4 插入查询结果
        insert into 表1(字段1，字段2，・・・) select(字段1，字段2，・・・) from 表2 where 条件; ――通过条件来定位哪一条数据
        将表2内指定条件查询出来的结果插入到表1中，注意查询出的数据要与指定的字段对应。

2.修改（更新）数据
  update 表名 set 字段1=值1,字段2=值2,・・・ where 条件1 and 条件2 and ・・・;
  同时修改多个值，用逗号分隔；通过条件来定位符合条件的数据条记录

3.删除数据
  3.1 删除指定的记录
        delete from 表名 where 条件; ――删除符合条件的一些记录
        注：
          当表内有自增id时，删除了一行数据，id仍然在之前的基础上继续自增（如删除了id=3的数据行后，再添加新的数据行时，id=4）
          若要自增id连续，则在新添加数据行之前，需执行  alter table 表名 auto_increment=1;
  3.2 清空所有记录（数据）
        delete from 表名; ――不加条件，则将所有内容都删掉
                 但是，一般我们用truncate ，能够将id置为零，delete不能将id置零，再插入的数据记录的id数继续递增。
        truncate 表名

4.查询数据
  select*from 表名; 	――查询所有数据
  select 字段1,字段2,・・・ from 表名; ――查询指定字段的数据

  详见单表操作、多表操作
